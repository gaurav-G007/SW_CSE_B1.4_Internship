Position property: dictates how an element is placed on a webpage. It controls whether an element stays in the normal page flow, or moves relative to its neighbour's or locks it in a fixed place when a user scrolls the page. 1. Static: Normal flow of the page (default position) 2. relative: Moves away from its normal spot without disturbing neighbours. 3.Absolute: Flies out of normal flow by positioning itself inside its closest relative parent 4. Fixed: Locks to the viewport. Stays in the exact same spot on the screen when scrolling. 5. Sticky: Scrolls naturally with the page, locks in place like FIXED

Both Display property and position property, controls layout, difference is that display creates the box and position moves the box
